# 等身大対話型ペッパーズ・ゴースト・システム（MVP）環境構築・起動手順書

本システムは、FastAPI（バックエンド）とThree.js/Web VRM（フロントエンド）を組み合わせ、最新の **Google GenAI SDK（gemini-2.5-flash）** と edge-tts による、超低遅延な「等身大ペッパーズ・ゴースト対話システム」です。

最新バージョンでは、ロード時のハードウェア競合を100%回避する「遅延起動（Lazy Initialization）＆ 独立スレッド構造」を採用。OSの環境差異に左右されない「高精度OpenCV-DNN顔検出」と「Geminiマルチモーダル属性看破」がシームレスに融合し、ライブデバッグモニターを搭載した商用クオリティの自律型アバターインフラに進化しています。

## 2. 【初回のみ】環境構築手順

新規PCでのセットアップ時、または環境再構築時のみ以下の手順を実行します。

### ① コマンドプロンプトを起動してディレクトリへ移動
```cmd
cd C:\Users\souda\workspace\peppers_ghost_avator
```

### ② Python仮想環境（venv）の作成
```cmd
python -m venv venv
```

### ③ 仮想環境の有効化（アクティベート）
```cmd
venv\Scripts\activate
```
※成功するとプロンプトの先頭に `(venv)` と表示されます。

### ④ 最新の依存ライブラリを一括インストール
リポジトリルートに配置された `requirements.txt` を用いて、次世代の `google-genai` を含む必須パッケージ群を一括導入します。
```cmd
pip install -r requirements.txt
```

## 3. 【運用時】日常の起動手順

環境構築が完了したPCで、日々システムを運用する際の手順です。

### ① コマンドプロンプトを起動し仮想環境に入る
```cmd
cd C:\Users\souda\workspace\peppers_ghost_avator
venv\Scripts\activate
```

### ② Gemini APIキーの環境変数セット
```cmd
set GEMINI_API_KEY=あなたの実際のAPIキー
```
※PowerShell環境の場合は `$env:GEMINI_API_KEY="あなたのAPIキー"` を使用してください。

### ③ ローカルサーバー（バックエンド）の起動
```cmd
python app.py
```
ターミナルに `Application startup complete.` と表示されれば待機状態になります。

### ④ Webブラウザ（Google Chrome）でアクセス
アドレスバーに以下のURLを入力してアクセスします（ルートマウントにより `/static` などの追記は不要です）。
```text
http://localhost:8000/
```

### ⑤ システムの本番点火
1. 画面右側にアバターが、左下にシアン色の明滅で「システム起動準備完了」と表示されるのを確認します。
2. **【Space】キー**（またはエメラルド色の起動ボタン）を押します。
3. これにより、ブラウザのマイクがON（赤いインジケータ点灯）になると同時に、バックエンド側で**カメラの電源が安全に遅延起動**し、人感センサーおよびライブモニターの配信が回り始めます。

## 4. 自律型インタラクティブ機能（AIの生命感演出）

* **高精度OpenCV-DNN ＆ Gemini属性看破お声がけ**
  OpenCVのディープラーニング顔検出（SSD/ResNetベース）により、横顔や影、照明の変化を物ともせず、前に立った人間を確実に捕捉。検知した瞬間の1フレームをGeminiのマルチモーダル推論へパスし、相手の「年齢層・性別・服装の色やアイテム・全体の雰囲気」を瞬時に看破。完全にパーソナライズされた固有のウェルカム挨拶をアバター自ら話しかけます（連呼防止のため、一度発話すると40秒間の冷却時間が入ります）。
* **ライブデバッグモニター（カメラプレビュー）**
  ハードウェアの競合クラッシュを防ぐため、Python側で取得したカメラ映像をWebSocket経由（`camera_preview` タイプ）でフロントに毎秒約10FPSで軽量ストリーミング配信。画面左上端にサイバーなインジケータ窓としてリアルタイム表示されます。
* **「気まぐれな独り言」の動的ランダムタイマー**
  完全に誰も話しかけない静寂な時間が続いた際、固定値ではなく**「60秒をベースに±30秒（30秒〜90秒）の範囲」で、会話が終わるたびに次回の呟きタイムアウト秒数をランダム計算**。50種類のフレーズプールから、その時の時間帯や投影環境（実機ホログラム筐体かPCモニターか）をメタに察知し、人間らしい気まぐれなテンポで独り言を呟きます。
* **設定変更リアクション**
  `Esc` キーでオプション画面を開き、声色やパラメータを調整して画面を閉じた際、アバターが設定を調整してくれたことへの可愛いリアクションをその場で1文喋ります。

## 6. トラブルシューティング（運用時のFAQ）

* **カメラのアクトランプが点灯しない / 属性看破で喋らない**
  * システムのロード直後は、競合フリーズを防ぐためカメラはあえて起動していません。必ず画面をロードしきった後に**【Space】キーを押して本番起動**を行ってください。
  * 起動してもランプがつかない場合は、ノートPCの内蔵カメラ等に割り当てられている可能性があります。`Esc` キーで設定画面を開き、**「カメラデバイス選択」を「カメラ 1」や「カメラ 2」に切り替えて**様子を見てください。
* **音声合成で `Invalid pitch` エラーがサーバー側に出る**
  * edge-ttsの仕様上、話速は `%` ですがピッチは絶対値（`Hz`）で指定する必要があります。UIスライダーからは自動的に `+30Hz` などの正しい型でパッキングされるよう対策済みです。
* **画面上の文字がすべて反転していて設定画面が読めない**
  * 本システムは等身大ホログラム筐体（ハーフミラー反射投影）を前提に、最初からCSSで画面全体を左右反転させています。通常のモニターでデバッグ・調整を行う際は、`Esc` メニュー内の**「画面左右反転モード」を「反転 OFF」**にしてください。
